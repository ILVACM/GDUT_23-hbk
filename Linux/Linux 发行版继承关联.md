## 一、CPU 架构基础

> | 信息项  | 内容         |

> | 更新日期 | 2026-03-01 |

选择 Linux 发行版前，首先需要了解 CPU 架构。不同架构对应不同的发行版版本。

| 架构        | 常见代号    | 通俗比喻           | 适合什么设备            | 你大概率会用到           |
|-----------|---------|----------------|-------------------|-------------------|
| AMD/Intel | x86_64  | 普通电脑的"标准汽油车"   | 90% 的笔记本/台式机/普通服务器 | ✅ 你大概率要选这个！       |
| ARM       | aarch64 | 手机/平板的"省电电动车"  | 手机、平板、树莓派、云服务器    | ✅ 用手机/树莓派就选它      |
| PowerPC   | ppc64le | 老式苹果 Mac 的"豪华跑车" | 旧苹果 Mac、IBM 服务器     | ❌ 除非你有 IBM 服务器，否则不用 |
| IBMZ      | s390x   | 银行核心系统的"巨无霸卡车" | 银行/政府大型机（处理百万笔交易） | ❌ 除非你是银行 IT，否则不用   |
| RISC-V    | riscv64 | 开源乐高积木的"未来概念车" | 新型物联网设备、定制芯片      | ⚠️ 小众，未来潜力大       |

### 3.2 发行版特点说明

| 发行版         | 发行版特点                                                          |
|-------------|----------------------------------------------------------------|
| Ubuntu      | 桌面友好、社区活跃、LTS 版本每 2 年更新（如 24.04 LTS）、云服务支持完善                       |
| Debian      | 稳定性标杆、软件包最全、分为 stable/testing/unstable 分支、企业级服务器常用               |
| Zorin OS    | 专为新手设计、模仿 Windows/macOS 界面、轻量级（基于 Ubuntu）、预装常用软件                  |
| Deepin      | 国产化桌面、美观的 UI 设计、内置中文生态、支持 Wine 运行 Windows 软件（基于 Debian）               |
| CentOS      | 曾是 RHEL 免费克隆版、企业服务器常用、2024 年 6 月 30 日停止维护（原计划 2024 年 12 月 31 日停更）           |
| AlmaLinux   | RHEL 1:1 兼容替代品、免费开源、替代 CentOS 8/9、企业级稳定（基于 RHEL 源码）               |
| Rocky Linux | RHEL 1:1 兼容替代品、社区驱动、替代 CentOS 8/9、企业级稳定（与 AlmaLinux 竞争）        |
| RHEL        | 商业企业级系统、付费订阅、10 年安全支持、严格测试、服务器/云平台首选（如 AWS/Azure）               |
| Fedora      | RHEL 试验田、新技术先行者、每 6 个月更新、软件最新、适合开发者和爱好者、社区驱动（Red Hat 赞助）       |
| openEuler   | 国产企业级系统、华为开源贡献、面向服务器/云计算、ARM/x86 多架构支持、国内生态完善、免费开源              |
| Linux Mint  | 基于 Ubuntu LTS、开箱即用（预装浏览器/多媒体/Flash 等）、Cinnamon 桌面优雅易用、适合 Linux 新手 |
| Robolinux   | 基于 Ubuntu LTS、预装 VirtualBox 并支持无缝运行 Windows 应用、适合需要虚拟化支持的用户         |

## 四、发行版分类体系

> | 信息项  | 内容         |

> | 创建日期 | 2026-03-01 |
> | 更新日期 | 2026-03-01 |

Linux 发行版主要根据软件包格式和包管理方式进行分类，这决定了它们的生态系统和使用体验。

| 分类        | 包格式         | 包管理器     | 代表发行版                                                                   | 主要特点                        |
|-----------|-------------|----------|-------------------------------------------------------------------------|-----------------------------|
| Debian 系列  | .deb        | APT/dpkg | Ubuntu, Debian, Linux Mint, elementary OS, MX Linux, Deepin, Kali Linux | 软件更新较快，适合桌面系统和开发环境，社区支持丰富   |
| Red Hat 系列 | .rpm        | YUM/DNF  | RHEL, Fedora, CentOS, CentOS Stream, openSUSE Leap, Rocky Linux         | 稳定性高，适合企业级服务器，软件更新较慢但经过严格测试 |
| Arch 系列    | .pkg.tar.xz | Pacman   | Arch Linux, Manjaro, EndeavourOS                                        | 高度可定制，适合技术爱好者，滚动更新          |
| Gentoo    | 源代码         | Portage  | Gentoo                                                                  | 基于源代码分发，需要编译，性能优化最佳         |

## 五、Linux 包格式详解

> | 信息项  | 内容         |

> | 更新日期 | 2026-03-01 |

| 包格式          | 代表发行版              | 管理方式         | 适合场景    | 优点         | 缺点         |
|--------------|--------------------|--------------|---------|------------|------------|
| .deb         | Ubuntu、Linux Mint  | apt/dpkg     | 桌面系统、新手 | 安装简单，社区支持好 | 依赖关系有时复杂   |
| .rpm         | RHEL、CentOS、Fedora | yum/dnf      | 企业服务器   | 稳定性高，企业支持好 | 依赖关系严格，更新慢 |
| AppImage     | 任意                 | 直接运行         | 便携应用    | 跨发行版，无需安装  | 体积较大       |
| Snap/Flatpak | 任意                 | snap/flatpak | 多版本共存   | 依赖隔离好，安全   | 体积更大，启动稍慢  |

## 六、镜像种类与应用

> | 信息项  | 内容         |

> | 创建日期 | 2026-03-01 | 
> | 更新日期 | 2026-03-01 |

| 镜像名称           | 定义作用                                       | 面向场景                         | 特点                                                                  |
|----------------|--------------------------------------------|------------------------------|---------------------------------------------------------------------|
| 默认镜像           | 最基础的安装介质，用于在物理机/虚拟机上安装系统                   | 服务器/虚拟机安装（VMware/VirtualBox） | • 体积小（Minimal ISO 约 1GB）<br>• 仅包含基础系统（命令行、基本工具）<br>• 适合服务器环境，无需图形界面 |
| desktop        | 包含图形界面（GNOME）的 默认镜像                        | 普通电脑/家庭用户/学生                 | • 包含浏览器、办公软件等<br>• 必须安装到硬盘才能使用<br>• 不能直接运行（需安装过程）                   |
| server         | 默认镜像的服务器版本，无图形界面                           | 服务器/云环境/开发测试                 | • 纯命令行界面，体积小<br>• 适合服务器部署<br>• 等同于 Rocky 的默认镜像                      |
| 云镜像            | 专为云平台（AWS/Azure/阿里云）优化的镜像，通常为 qcow2/vhd 格式 | 云服务器部署                       | • 包含云平台所需工具（如 cloud-init）<br>• 直接上传云平台即可使用<br>• 无需额外配置，省时省力         |
| Docker 镜像（OCI）  | 用于 Docker 容器的轻量级镜像，专为容器化应用设计               | 容器化应用部署（Docker/Kubernetes）   | • 体积小（100MB-500MB）<br>• 通过 `docker pull` 命令获取<br>• 避免系统臃肿，适合微服务架构   |
| 桌面/工作站 Live 镜像 | 包含完整桌面环境的镜像，可直接启动试用无需安装                    | 个人电脑体验/桌面系统                  | • 包含图形界面（GNOME）和常用应用<br>• 可直接启动试用（U 盘启动）<br>• 体积较大（4-6GB），适合体验而非安装   |
| WSL 镜像          | 专为 Windows Subsystem for Linux 优化的镜像       | Windows 系统上运行 Linux          | • 专为 WSL 设计，安装简单<br>• 无需额外配置，双击即可使用<br>• 体积适中（500MB-1GB）            |
